# ACED

A roguelike-deckbuilder study engine. Drop in your own study materials — textbook excerpts, practice questions, exam blueprints — and ACED turns them into a Balatro-style run with cards, jokers, boss blinds, and score multipliers.

Designed to work for any large test: CPA, Bar, MCAT, USMLE, GRE, CFA, NCLEX, AWS, AP, and so on.

## Running locally

Open `index.html` in a modern browser. No build step.

## Modes

- **Pack-driven run** (`study.html`) — the v6 engine. Cards, jokers, ×Mult snowball.
- **Generic engine** (`engine.html`) — alternate run mode with study-technique jokers.
- **Importer** (`importer.html`) — paste study material → generate a pack with your own LLM key.
- **Library** (`library.html`) — browse built-in packs.

## Built-in packs

- `packs/cpa-far.js` — CPA FAR concept pack (original content)
- `packs/far-sims.json` — FAR simulations pack
